## Sơ đồ vận hành một khách (runbook)

```
1. LEAD (BD)          A0 → qualify → WeChat/CRM
2. KHẢO SÁT (PM)      A1/A2 → Site selection → Pre-FS phụ lục
3. HỢP ĐỒNG           Full Setup / B2+B1 → deposit 30%
4. PHÁP LÝ            IRC → ERC → sub-license (ĐTM/PCCC/EPE)
5. LOGISTICS          HS code → tờ khai #1 → container #1 vào NM
6. BÀN GIAO           Milestone M6 + C1 retainer offer
```
