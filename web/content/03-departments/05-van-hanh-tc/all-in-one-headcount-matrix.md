# All-in-One — Cơ cấu nhân sự

> **Đặc thù khách TQ:** Nhanh · tin cậy · **không rào cản ngôn ngữ** (HSK 5–6, Guanxi).  
> **Nguyên tắc LONGTV:** Giai đoạn 1 **lean + đối tác**; giai đoạn 2–3 tiến tới bảng nhân sự đầy đủ bên dưới.

---

## 1. Bảng nhân sự MỤC TIÊU (Y3 — vận hành trơn tru)

*Hồ sơ gọi vốn / operational manual đầy đủ — chưa tuyển hết năm 1.*

| Bộ phận | Vị trí | Số | Yêu cầu năng lực cốt lõi |
|---------|--------|-----|---------------------------|
| **Ban điều hành** | Giám đốc Dự án (Project Director) | 1 | >5 năm KCN / tư vấn FDI; am hiểu Ban QLKCN HP & TN |
| **Pháp lý** | Luật sư / CV pháp lý chính | 2 | Luật ĐT, DN; PCCC & môi trường |
| | CV giấy phép phụ (sub-license) | 2 | ĐTM, GP chuyên ngành |
| **Logistics** | CV hiện trường (Ops) | 3 | HQ hiện trường, kiểm tra chuyên ngành |
| | CV chứng từ (Docs) | 2 | **Chứng chỉ nghiệp vụ HQ** TCHQ |
| **BD / Khách hàng** | CV phát triển thị trường | 3 | **HSK 5–6**; Guanxi; ưu tiên Hoa kiều / du học TQ |
| **Hỗ trợ** | Kế toán, Admin, IT | 2–3 | — |
| **TỔNG** | | **~16–18 FTE** | |

*Luật sư hành nghề có thể nằm trong Công ty Luật đối tác — không nhân sự LONGTV.*

## 2. Giai đoạn BOOTSTRAP (Y1 — vốn 2 tỷ, đang chạy)

| Bộ phận | Vị trí Y1 | Số | Cách lấp đầy |
|---------|-----------|-----|--------------|
| Ban ĐH | **TGĐ** (kiêm Project Director) | 1 | Sếp Thắng |
| | PM dự án Full Setup | 1 | Kiêm / tuyển T10 |
| Pháp lý | PM pháp lý (không GPLS) | 0.5–1 | Nội bộ + **MOU luật** |
| | Luật sư | 0 | **Đối tác** |
| Logistics | Coordinator HQ | 1 | Kiêm + **MOU đại lý** |
| | Ops / Docs | 0 | **Đối tác** |
| BD | Trưởng nhóm KD | 1 | Tuyển T8 |
| | Sales Exec song ngữ | 1 | Tuyển T9 |
| | CV BD thứ 3 | 0 | Y2 khi pipeline >10 |
| Hỗ trợ | Admin / kế toán thuê ngoài | 0.5 | Part-time |
| **TỔNG FTE tương đương** | | **8–10** | [org-model](/docs/03-departments/00-org-model) |

**Gap Y1 → Y3:** ~8–10 FTE — chủ yếu Logistics Ops/Docs và luật in-house (nếu mở văn phòng luật liên kết).

## 5. JD tóm tắt — 3 vị trí ưu tiên tuyển

### 5.1. Sales Executive (TQ) — P0

- HSK 5+; outbound WeChat / LinkedIn
- KPI: 20 outreach/tuần, 3 meeting/tháng
- JD đầy đủ: KIM-073 (C-level pack)

### 5.2. PM Dự án All-in-one — P1 (T1/2027)

- 3+ năm FDI / KCN; song ngữ
- Own timeline 12 tuần Full Setup
- RACI: [operational-manual](/docs/07-operations/all-in-one-operational-manual)

### 5.3. CV Chứng từ Hải quan — P2 (T2/2027)

- Chứng chỉ nghiệp vụ TCHQ
- HS code máy móc TSCĐ; phối hợp ECUS đối tác
