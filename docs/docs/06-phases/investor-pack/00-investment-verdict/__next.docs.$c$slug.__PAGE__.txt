<li><strong>Pivot sớm nếu:</strong> 0 Won sau 90 ngày + pipeline &#x3C;5 qualified → xem <a href="/docs/04-research/2026-07/plan-b-no-revenue">Plan B</a></li>
</ol>
<h3>Không nên (giai đoạn 1)</h3>
<ul>
<li>Thuê văn phòng riêng >15M/tháng trước deal đầu</li>
<li>Tuyển sales thứ 3 trước utilization >70%</li>
<li>Bán A3 (thiết lập NM full) trước uy tín + đối tác đất</li>
<li>Hứa investor ROI 12 tháng — mô hình consulting cần 18–24 tháng</li>
