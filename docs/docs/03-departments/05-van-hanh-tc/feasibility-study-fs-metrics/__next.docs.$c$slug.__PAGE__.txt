<h2>2. NPV (Net Present Value)</h2>
<pre><code>NPV = −800 + Σ [FCF_t / (1,12)^t]

t=0,5: −210 / 1,057 ≈ −199
t=1:   +153 / 1,12   ≈ +137
t=2:   +710 / 1,254   ≈ +566
t=3:   +1510 / 1,405  ≈ +1075
t=4:   +2410 / 1,574  ≈ +1531

NPV ≈ −800 −199 +137 +566 +1075 +1531 ≈ **+2.310 triệu VND**
</code></pre>
<p><strong>NPV trên vốn triển khai 800M @12%:</strong> <strong>~+420 đến +2.310 triệu</strong> tùy cách gộp Y0.</p>
<p><strong>Báo cáo investor (conservative):</strong> <strong>NPV +~400–500 triệu VND</strong> (chỉ 3 năm đầu, discount 15%).</p>
