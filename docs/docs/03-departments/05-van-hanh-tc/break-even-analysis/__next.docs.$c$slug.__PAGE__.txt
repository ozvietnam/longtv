<h2>2. Hòa vốn tháng (full lean)</h2>
<p><strong>OPEX = 100M/tháng</strong> (làm tròn)</p>
<pre><code>Doanh thu cần = OPEX / GM = 100M / 0.58 ≈ 172M VND/tháng
               ≈ $6.900 USD/tháng
</code></pre>

<h2>3. Hòa vốn bootstrap (T7–T9)</h2>
<p><strong>OPEX = 65M/tháng</strong></p>
<pre><code>Doanh thu cần = 65M / 0.58 ≈ 112M VND/tháng ≈ $4.500/tháng
</code></pre>
<p>→ <strong>~1,4 A1/tháng</strong> hoặc <strong>1 A1 + retainer nhỏ</strong>.</p>
<p>Giai đoạn đầu <strong>chưa có 2 sales full</strong> — hòa vốn không phải mục tiêu T7–T8; mục tiêu là <strong>1 Won trong 60 ngày</strong>.</p>
